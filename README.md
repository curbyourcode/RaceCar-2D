# RaceCar 2D game
A 2D car game with Angular. You start with a car and have to get to the end of the track without hitting another vehicle. Very simple practical game used car sprites.

## How to Use this appliaction tutorial
1. Clone down repo
2. Run npm install
3. Run ng serve

## Technologies Used
Angular cli 7
sprites
